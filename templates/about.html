About Me

Welcome to your personal library—an organized space where you can store an unlimited number of books and authors. With our platform, you can easily manage and connect books to their authors, ensuring everything stays neat and accessible in one place. The intuitive search feature allows you to find exactly what you need in no time.

¿Why this project?

This project is part of the second trimester for the Web Application Implementation module. The goal is to create a web application using Flask that fulfills the following requirements:

  • Use a Git repository
  • Create a virtual environment
The core features include:
  • A list view to display the items in the library
  • A search function to find books and authors quickly
  • A detailed view for each item
  • CRUD operations for managing books and authors
  • An Entity-Relationship (E-R) model with at least three entities
Additionally, optional features include:
  • User management (login and sign-up)
  • Integration of a CSS framework for styling
